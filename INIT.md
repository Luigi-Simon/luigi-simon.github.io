Project initialization notes
---------------------------

This repository contains a simple static personal site (HTML/CSS/JS).

Quick start

- Using Python (no install needed):

```powershell
python -m http.server 8000
```

- Using npm (uses `http-server` via npx):

```powershell
npm run start
```

Created files

- `.gitignore` — ignores node_modules and common files
- `package.json` — adds convenient `start` scripts
